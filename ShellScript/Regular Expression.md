# Regular Expression
## 1. 정규표현식

> 다른 문자열을 검색하거나 지환할 목적으로 고안된 특수 문자열

- grep, sed, awk 등 유닉스 명령행 도구 중 일부는 정규표현식 사용
- [문자] : 대괄호 안 문자들을 검색
  - [a-zA-Z0-9] : 모든 알파벳, 숫자
- [^문자] : not
  - [^a-zA-Z0-9] : 모든 알파벳, 숫자를 제외한 나머지
- \\w : 모든 글자 (=[a-zA-Z0-9])
- \\W : 모든 알파벳, 숫자 제외한 나머지

- .(dot) : 모든 글자 (공백 포함)

- \+ : 문자의 반복
  - ^[.+]\(공백) : 단어+공백
  - n+ : 'n'의 반복

- \\s : 공백 문자
- \\S : 공백문자가 아닌 문자

- \d : 숫자
- \D : 숫자가 아닌 문자

- \\b : 바운더리
  - \\b.+?\\b : 단어만을 검색

- ? : 0~1 글자
  - http?:\/\/.+ : "http://", "https://" 로 시작하는 문자 검색

- () : 그룹 선택
  - http?:\/\/(www\\.)?.+ : "http://", "https://" 로 시작 & "www."이 포함되어도, 안되어도 검색
