## 2. #define
- #define 전처리 문장을 매크로라고 한다.
- 매크로 상수, 매크로 함수로 나눌 수 있다.

### 1) 매크로 상수
```c
#define [매크로명] [값]
```

```c
// ex
#define PI 3.1415
#define MAX 10000
```

- #definde문 끝에는 세미콜론을 붙이지 않는다.
- 매크로 **상수**의 값은 변경할 수 없다. 
- 함수명이나 데이터형도 정의할 수 있다.
```c
#define PRT printf
PRT("Hello World!);

#define UINT unsigned int
UINT number;
```

### 2) 매크로 함수
```c
#define [매크로 함수명] (인자) [내용]
```

```c
// ex
#define SQUARE(a) a*a
int sq = SQUARE(2);
// int sq = 2*2; 와 같다
```
