# Compiler / Interpreter

- 컴파일러와 인터프리터 모두 C나 Java같은 고레벨언어를 기계어로 번역한다.
- 속도 : 컴파일러 > 인터프리터
- C는 컴파일러 언어, 파이썬은 인터프리터 언어고 Java는 둘 다 사용한다.


### Compiler
- **전체 코드**를 보고 번역한다.
- 코드를 바로 기계어로 변환한다.
- 메모리를 많이 쓴다.


### Interpreter
- 코드의 **각 줄**마다 번역한다.
- 기계어로 바로 변환하지 않고 중간 단계를 거친다.
- 메모리를 적게 쓴다.
